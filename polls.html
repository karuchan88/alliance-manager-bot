Poll System

The poll feature allows leaders and members to create polls within their alliance.

the available options for answers include Yes/No/Maybe and are more customized for title calls.

A great use for this feature is to create a poll for title calls and allow members to vote on if they will be going for the title of that specific event or not.

The polls also provide a proxy vote option for members who aren't in discord but still want to vote.

The polls also have a timer function that can be enabled to automatically end the poll after a certain amount of time.

Any normal players attempting to vote for a locked poll will receive a message that the poll is locked.

Additional roles of the bot include Namekeeper Manager and Proxy Votes.

Namekeeper Manager of course allows the user assigned to manage the bot and update nicknames and MVP.

Proxy Votes allows the assigned user to vote on behalf of members who aren't in discord but still want to vote.

Only the Namekeeper Manager role can override a locked poll and provide a proxy vote for a member.
