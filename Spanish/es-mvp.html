Seguimiento de MVP

Alliance Manager permite a los líderes registrar los MVP de los eventos estándar, el Ajedrez de Alianza y la Batalla de las Ruinas.
Actualmente, estas son las tres categorías disponibles para el seguimiento, aunque su uso no es obligatorio.
Los líderes pueden añadir registros de sus miembros, y el bot llevará un control de los MVP de cada miembro en cada categoría.
Esto permite a los líderes ver fácilmente quién ha sido votado como MVP en su alianza recientemente, facilitando la elección para el próximo evento.
También existe una función de bloqueo para aquellos miembros que deseen excluirse de las votaciones de MVP o que prefieran no participar por motivos personales. Se puede aplicar un bloqueo a cualquier miembro para que su nombre no aparezca en la lista de selección al registrar datos.

Instrucciones

/editmvp

Para registrar un nuevo MVP, utiliza el comando "/editmvp".

Tras introducir el comando, se pregunta primero por el tipo de evento; a continuación, se selecciona al miembro y, finalmente, se introduce la fecha de la votación.

Una vez introducida la información y enviado el mensaje, aparecerá un mensaje de confirmación.

El resultado final se muestra con un formato adecuado para publicarlo en un canal destinado a anuncios de MVP.

/blockmvp

Utiliza el comando "/blockmvp" para impedir que un miembro pueda ser seleccionado como MVP. Deberás introducir la información del jugador y también podrás añadir una nota.
Ten en cuenta que esto no bloquea al miembro en Discord.

Esto es solo para evitar posibles errores al elegir un MVP.

Tras introducir la información y enviar el mensaje, aparecerá un mensaje de confirmación.

Se muestra la confirmación final.

/blockedlist

Para ver la lista de miembros bloqueados, utiliza el comando "/blockedlist".

Una demostración de cómo aparece un jugador "bloqueado" al usar /mvplist.

/mvplist

Para ver la lista de registros de MVP, utiliza el comando "/mvplist".

Muestra la lista de registros de MVP.

/resetmvp

Para borrar los registros de MVP de todos los miembros de una categoría específica, utiliza el comando "/resetmvp". Esto eliminará todos los registros de esa categoría y la acción es irreversible; por lo tanto, utilícelo con precaución y asegúrese de usar archivos de respaldo si es necesario.

Tras introducir el comando, seleccione la categoría que desea restablecer.

Una vez seleccionada la categoría, aparecerá un mensaje de confirmación.

Tras confirmar, se mostrará el mensaje final que confirma el restablecimiento de dicha categoría.

/unblockmvp

Utilice el comando "/unblockmvp" para desbloquear a un miembro de las selecciones de MVP. Deberá introducir la información del jugador.

Tras introducir la información y enviar el mensaje, aparecerá un mensaje de confirmación.

Se muestra la confirmación final.

Notas adicionales de las pruebas:
Si existen registros de un jugador y posteriormente se le bloquea, dichos registros permanecerán en segundo plano pero no aparecerán en la lista de MVP. No obstante, estos registros desaparecerán si se utiliza un comando de restablecimiento.
Si no se realiza un restablecimiento y se desbloquea al jugador, tenga en cuenta que esos registros antiguos volverán a aparecer la próxima vez que se consulte la lista de MVP.
