Bot Commands

Here are the available commands for the Alliance Manager bot:

  • Initial Setup Commands:
  • /ihelp - Displays the help menu for brief descriptions and how to setup bot.
  • /setup - Initiates the bot setup process.
  • /setupstatus - Sets up or repairs the bot's status embed in the specified channel. This is the static message that shows whether the bot is currently active.
  • /synccommands - Owner-only command used to synchronize the bot's commands.
  • Name Management Commands:
  • /listnames - Lists all tracked nicknames for the server.
  • /addmember - Adds a member to the nickname tracking system. Requires the member's UID and their current in-game nickname.
  • /editmember - Edits a member's information in the nickname tracking system. Requires the member's UID and the updated information.
  • /deletemember - Removes a member from the nickname tracking system. Requires the member's UID.
  • /bulkadd - Add many members at once from a Google Sheets or Excel copy/paste.
  • /exportnames - Exports all tracked UIDs, Primary Names, Alt names, MVP fields to a CSV file.
  • MVP Tracking Commands:
  • /mvplist - Lists all tracked MVP records for the alliance.
  • /editmvp - Edits an existing MVP record for a member. Requires the member's UID, the category of the event, and the updated information.
  • /resetmvp - Clears the selected MVP category field for all members.
  • /blockmvp - Blocks a member from being selected for MVP.
  • /unblockmvp - Unblocks a member from MVP selections.
  • /blockedlist - Lists all members currently blocked from MVP selections.
  • Poll Commands:
  • /poll - Creates a new poll with the specified question.

How voting works:

  • Members vote using the poll buttons.
  • Votes do not appear immediately.
  • There is a 60 second delay before votes are shown/updated.
  • This helps prevent people from simply following early votes.

Proxy Votes:

Managers or members with the allowed proxy role can vote on behalf of others when needed.

Close Time:

When creating a poll, you can set when it closes.

Once closed, voting buttons should no longer accept new votes.
