Politique de confidentialité

Cette politique de confidentialité décrit comment vos informations personnelles sont collectées, utilisées et partagées lorsque vous utilisez le bot Alliance Manager.

Les données collectées incluent les identifiants de serveur, les identifiants de salon et toute information que vous soumettez au bot via ses commandes.

Les fonctionnalités du bot reposent sur les informations que vous fournissez pour les améliorations optionnelles de votre alliance. Ce bot n'est en aucun cas obligatoire pour les alliances ; son utilisation est laissée à votre entière discrétion.

Nous utilisons ces informations pour fournir et améliorer les fonctionnalités du bot, telles que le suivi des MVP, la gestion des surnoms et la création de sondages.

Nous ne partageons pas vos informations avec des tiers, sauf si cela est nécessaire pour fournir les services du bot ou pour se conformer à des obligations légales.

En utilisant le bot Alliance Manager, vous consentez à la collecte et à l'utilisation de vos informations telles que décrites dans cette politique de confidentialité.

Toutes les données sont stockées dans des fichiers JSON locaux jusqu'à nouvel ordre. Chaque serveur dispose de son propre fichier de données, tandis que les sondages fonctionnent séparément dans leur propre fichier. Si vous souhaitez que vos données soient supprimées, veuillez contacter l'administrateur du bot.

Les informations stockées dans le code ne sont consultées que lors de sessions de maintenance ou de débogage, ou suite à des demandes personnelles de suppression de données.

Si le bot devait être transféré vers une autre plateforme d'hébergement, des notifications seraient envoyées ; vous pouvez cesser d'utiliser le bot à tout moment.

Toutes les informations contenues dans le code du bot seraient transférées avec la plateforme d'hébergement ; si vous souhaitez qu'elles soient supprimées, vous pouvez en faire la demande à tout moment.

Veuillez ne pas soumettre d'informations privées ou de mots de passe au bot !
