Suivi des MVP

Alliance Manager permet aux chefs d'alliance de suivre les titres de MVP obtenus lors des événements standards, des Échecs d'alliance et de la Bataille des ruines.
Il s'agit actuellement des trois catégories disponibles pour le suivi ; leur utilisation est facultative.
Les chefs peuvent enregistrer les titres obtenus par leurs membres, et le bot conserve un historique des MVP pour chaque membre dans chaque catégorie.
Cela permet aux chefs de voir facilement quel membre a été élu MVP le plus récemment au sein de l'alliance, en vue du prochain vote.
Une fonctionnalité permet également aux membres de se retirer du système de vote MVP, que ce soit par choix personnel ou pour toute autre raison. Il est possible d'appliquer un blocage à n'importe quel membre ; son nom n'apparaîtra alors plus dans la liste de sélection lors de la saisie des enregistrements.

Instructions

/editmvp

Pour saisir un nouvel enregistrement MVP, utilisez la commande « /editmvp ».

Après avoir saisi la commande, la première étape consiste à choisir le type d'événement, puis à sélectionner le membre et enfin à indiquer la date du vote.

Une fois les informations saisies et le message envoyé, un message de confirmation s'affiche.

Le résultat final est alors affiché dans un format adapté à une publication dans un canal dédié aux annonces MVP.

/blockmvp

Utilisez la commande « /blockmvp » pour empêcher la sélection d'un membre en tant que MVP. Vous devrez saisir les informations du joueur ; il est également possible d'ajouter une note.
Remarque : cette action ne bloque pas le membre sur Discord. Ceci permet simplement d'éviter toute erreur lors de la sélection d'un MVP.

Une fois les informations saisies et le message envoyé, un message de confirmation s'affiche.

La confirmation finale s'affiche.

/blockedlist

Pour consulter la liste des membres bloqués, utilisez la commande « /blockedlist ».

Exemple d'affichage d'un joueur « bloqué » via la commande /mvplist.

/mvplist

Pour consulter la liste des enregistrements MVP, utilisez la commande « /mvplist ».

Affiche la liste des enregistrements MVP.

/resetmvp

Pour effacer les enregistrements MVP de tous les membres d'une catégorie spécifique, utilisez la commande « /resetmvp ».

Cette action supprimera toutes les données de cette catégorie de manière irréversible ; veuillez donc l'utiliser avec précaution et vous assurer de disposer de sauvegardes si nécessaire.

Après avoir saisi la commande, sélectionnez la catégorie à réinitialiser.

Une fois la catégorie sélectionnée, un message de confirmation s'affichera.

Après confirmation, le message final apparaîtra pour confirmer la réinitialisation de la catégorie.

/unblockmvp

Utilisez la commande « /unblockmvp » pour débloquer un membre concernant les sélections MVP. Vous devrez saisir les informations du joueur.

Après avoir saisi les informations et envoyé le message, un message de confirmation s'affichera.

La confirmation finale s'affiche.

Remarques supplémentaires issues des tests :
Si des données concernant un joueur ont été enregistrées avant que celui-ci ne soit bloqué, ces données resteront en mémoire mais n'apparaîtront pas dans la liste MVP. Elles disparaîtront toutefois si une commande de réinitialisation est utilisée.
Si aucune réinitialisation n'est effectuée et que le joueur est débloqué, sachez que ces anciennes données réapparaîtront lors de la prochaine consultation de la liste MVP.
